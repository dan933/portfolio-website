Fullstack Invoicing Application

A full stack invoicing application created with: Angular for the frontend, .NET for the API, and SQL server for the database. The user can perform CRUD operations on the customers and invoices. The data tables are paginated, can be filtered, and sorted. The user can download invoices in pdf format from the client. GST can be added and removed from the invoices. The source code for this project can be found here Invoice Application.


NBA Prediction Application
Group Project

A full stack application created with: React for the frontend, .NET for the API, Auth0 for authentication, SQL server for the database and azure piplines for CI/CD. This was a group project, collaboration was done through github. A user could login create an NBA dream team by adding or removing players from a team, a user was then able to compare two teams to see which team had a better win percentage. The source code for this project can be found here NBA Prediction Application.
