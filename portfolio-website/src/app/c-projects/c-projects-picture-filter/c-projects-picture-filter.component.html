Picture Filter

The picture filter project was part of Harvard's CS50 course problem set four. The program takes a .bmp file as an input and returns an output file with the users desired filter. My job in this problem set was to create the part of the program that manipulates the pixels and returns the desired filter.

Grayscale

Source Code 
            {{'// Convert image to grayscale
            void grayscale(int height, int width, RGBTRIPLE image[height][width])
            {

              //var that stores avg rgb value
              float avg_rgb_pxl;
              float b;
              float g;
              float r;

              // loop over height
              for (int i = 0; i < height; i++)
              {
                  // loop over width
                  for (int j = 0; j < width; j++)
                  {
                      //avg rgb pixel value
                      b = image[i][j].rgbtBlue;
                      g = image[i][j].rgbtGreen;
                      r = image[i][j].rgbtRed;

                      avg_rgb_pxl = (b + g + r) / 3 + 0.5;

                      //convert rgb to grayscale
                      image[i][j].rgbtBlue = avg_rgb_pxl;
                      image[i][j].rgbtGreen = avg_rgb_pxl;
                      image[i][j].rgbtRed = avg_rgb_pxl;
                  }
              }
              return;
            } '}}
