Picture Filter

The picture filter project was part of Harvard's CS50 course problem set four. The program takes a .bmp file as an input and returns an output file with the users desired filter. My job in this problem set was to create the part of the program that manipulates the pixels and returns the desired filter.

Grayscale

To transform an image to grey the red, green and blue values in the pixel have to be the same. To achieve this I used a nested for loop to iterate over every pixel captured the red, green and blue values and calculated the average of that pixel. I then assigned that value to the red, green and blue values in that pixel.
